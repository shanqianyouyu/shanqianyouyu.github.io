redis的简单使用
Redis
数据库
发布日期:   2019-08-05
文章字数:   1.1k
阅读时长:   4 分
阅读次数:  

Redis：C语言编写的Nosql数据库(非关系数据库),数据保存在内存中。

特点：

  • 数据保存在内存中
  • 不定期持久化，保证数据安全
  • 跨语言
  • 支持集群
mysql redis memcache
类型 关系型 非关系型 非关系型
存储位置 磁盘 磁盘和内存 内存
存储过期 不支持 支持 支持
读写性能 低 非常高 非常高

redis支持的类型比memcache多

##使用场景

  • 缓存(**可以做mybatis的二级缓存)

  • 计数器应用

    • 网站需要统计注册用户数，网站总浏览次数
    • 新浪微博转发数，点赞数
      (能自增，自减)

  • 实时防攻击系统：

    • 防暴力破解：试密码

  • 排行榜

  • 设定有效期(自动过期)
  • 自动去重应用(set数据结构)
  • 队列
    • 秒杀
  • 消息订阅系统

结构

  • redis-benchmark.exe ：性能测试工具
  • redis-cli.exe ： 客户端
  • redis-server.exe : 服务端
  • redis-check-aof.exe : 检测持久化文件完好
  • redis.windows.conf : 配置文件

redis使用

启动redis服务：

带配置文件启动

redis-server.exe redis.windows.conf

关闭窗口则会关闭redis

连接到redis

cmd > {redis路径} redis-cli ip地址 -p 端口号
ip默认本地，-p默认6379

redis-cli.exe -h 127.0.0.1 -p 6379

基本用法

#选中数据库，默认有0-15号数据库
select num

#设置一个key对应一个value
set key value

#取一个key对应的值
get key

#设置一个key的过期时间
expire key seconds

#查看一个key的剩余过期时间
ttl key

#查看所有的key
keys *

#设置多个key-value
mset key1 value1 key2 value2

#根据key返回value
mget key1 key2

#将key中存储的数据增1(key不存在，则初始化为0，再加1)
incr key

#将key中存储的数字值减1(key不存在，则初始化为0.再减1)
decr key

incrBy key
decrBy key

# 删除key
del key

#清除所有数据库
flushall

#清除当前库
flushdb

设置密码

在配置文件添加

#此处密码设置为chase
requirepass chase

登录校验密码：

#密码认证
auth chase

配制文件

以前没注意过，先给出。

# 1k => 1000 bytes
# 1kb => 1024 bytes
# 1m => 1000000 bytes
# 1mb => 1024*1024 bytes
# 1g => 1000000000 bytes
# 1gb => 1024*1024*1024 bytes

#设置秒数，超过时间不操作则断开连接，0为不会断开
timeout 0 

#日志级别
loglevel notice

#日志文件位置 当前文件夹下logs文件夹下的redis.log文件下
logfile ./logs/redis.log

#设置数据库数量
databases 16


#持久化策略，900秒有一次改变就持久化一次，300秒内有10次改变就持久化一次，60秒有10000次改变就持久化一次
save 900 1
save 300 10
save 60 10000

#持久化文件的位置
dbfilename dump.rdb


#是否开启持久化
appendonly no

对list集合的常用操作

list看成是左右排列的集合

#向列表中从左边推送两个值
lpush key value1
lpush key value2

#从左边显示 start 到 stop 的值，stop 为-1表示最后一个
lrange key start stop

#左边弹出一个值
lpop key

#根据count值移除列表key中与参数value对应的元素
lrem key count value
#exp:
#删除student中，从左边数的第一个 zs
lrem stuents 1 zs
#删除student中，从左边数的前两个 zs
lrem stuents 2 zs
#删除student中，所有的张三
lrem stuents 0 zs
#删除student中，从右边数的第一个 zs
lrem stuents -1 zs
#删除student中，从右边数的前两个 zs
lrem stuents -2 zs

Jedis操作redis

简单直接操作

导包:

  • commons-pool2-2.2.jar
  • jedis-2.5.2.jar

    @Test
    void testSet() {
        // 创建连接
        Jedis connection = new Jedis("127.0.0.1", 6379);
        //登录密码
        connection.auth("chase");
        // 执行连接
        connection.set("name", "zs");
        // 关闭连接
        connection.close();
    }

    @Test
    void testGet() {
        // 创建连接
        Jedis connection = new Jedis("127.0.0.1", 6379);
        connection.auth("chase");
        // 执行连接
        System.out.println(connection.get("name"));
        // 关闭连接
        connection.close();
    }
