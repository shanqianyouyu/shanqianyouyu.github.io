Linux常用指令

一. 文件和目录

1. cd

2. ls

3. cp

用于复制文件，copy之意，它还可以把多个文件一次性地复制到一个目录下

4. mv

用于移动文件、目录或更名

5. rm

二. 查看文件

6. cat命令

三. 文件搜索

7. find

四. 文件的权限 - 使用 “+” 设置权限，使用 “-“ 用于取消

8. chmod

9. chown

改变文件的所有者

10. chgrp

改变文件所属用户组

五. 文本处理

11. grep

12. paste

13. sort

14. common

六. 打包和压缩文件

15. tar

对文件进行打包，默认情况并不会压缩，如果指定了相应的参数，它还会调用相应的压缩程序（如gzip和bzip等）进行压缩和解压

七. 系统和关机（关机、重启和登出）

八. 进程相关的命令

16. jps

显示当前系统的java进程情况，及其id号
jps(Java Virtual Machine Process Status Tool)是JDK 1.5提供的一个显示当前所有java进程pid的命令，简单实用，非常适合在linux/unix平台上简单察看当前java进程的一些简单情况。

17. ps

用于将某个时间点的进程运行情况选取下来并输出，process之意

18. kill

用于向某个工作（%jobnumber）或者是某个PID（数字）传送一个信号，它通常与ps和jobs命令一起使用

19. killall

向一个命令启动的进程发送一个信号

20. top

是Linux下常用的性能分析工具，能够实时显示系统中各个进程的资源占用状况，类似于Windows的任务管理器

21. 如何杀死进程：

  • 图形化界面的方式
  • kill -9 pid （-9表示强制关闭）
  • killall -9 程序的名字
  • pkill 程序的名字

22. 查看进程端口号

netstat -tunlp|grep 端口号
