vim入门
Linux
Linux相关

文章字数:   620
阅读时长:   2 分
阅读次数:  

早就听说Vim是宇宙第一编辑器，之前只会保存编辑，很多功能都不会，今天抽了点时间学了一下指令，记录一下。

使用

三种模式

  1. 普通模式：Esc
  2. 插入模式：i或者a或者c
  3. 可视模式：
    • 普通可视 v
    • 行可视 V
    • 块可视 ctrl+v

命令

普通模式下：

  • h 左移
  • j 下移
  • k 上移

  • l 右移

  • u 撤回

  • gg 回到行首

单词移动

普通模式下：

  • w 正向移动到下一单词的靠头
  • b 反向移动到当前单词/上一单词的开头
  • e 正向移动到当前单词/下一单词的结尾
  • ge 反向移动到上一单词的结尾

字符移动

f{char} 正向移到下一个{char}所在之处
F{char} 反向移到下一个{char}所在之处
t{char} 正向移到下一个{char}所在之处的前一个字符上
T{char} 反向移到下一个{char}所在之处的后一个字符上

; 重复上次字符查找命令
, 反转方向查找上次字符的查找命令

文本对象

i-in a-around

  • a) 或 ab 一对圆括号
  • i) 或 ib 圆括号内部
  • at 一对XML标签\<xml>tags\<\/xml>
  • it xml标签内部
  • iw 当前单词
  • aw 当前单词加一个空格
  • iW 当前字串
  • aW 当前字串加一个空格
  • is 当前句子
  • as 当前句子及一个空格
  • ip 当前段落
  • ap 当前段落及一个空行

操作符待决模式

d{motion} 删除模式 dd删除一行
c{motion} 修改模式 cc删除一行
y{motion} 复制模式 yy复制一行
v{motion} 可视模式 -

标记

  • m{mark} 设置标记
  • `{mark} 返回标记(esc下面的键位)

翻页

  • ctrl+f 下翻一页
  • ctrl+b 上翻一页
  • ctrl+d 下翻半页
  • ctrl+u 上翻半页

查找与替换

/{pattern} 查找 使用n跳转
:%s/{pattern}/{string}/g 替换 使用c进行替换确认

配置

这里配置只是为了文本好看，就没有搞花里胡哨的，或者添加别的功能。

vim ~/.vimrc

set tabstop=4
set softtabstop=4
set shiftwidth=4
set autoindent
set cindent
set cinoptions={0,1s,t0,n-2,p2s,(03s,=.5s,>1s,=1s,:1s
set nu
set ruler
set hlsearch
set backspace=2
