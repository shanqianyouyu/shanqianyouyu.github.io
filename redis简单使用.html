redis简单使用 

> Redis：C语言编写的Nosql数据库(非关系数据库),数据保存在内存中。

###
特点：
* 数据保存在内存中
* 不定期持久化，保证数据安全
* 跨语言
* 支持集群

|      | mysql | redis | memcache |
|------|-------|-------|----------|
| 类型   | 关系型   | 非关系型  | 非关系型     |
| 存储位置 | 磁盘    | 磁盘和内存 | 内存       |
| 存储过期 | 不支持   | 支持    | 支持       |
| 读写性能 | 低     | 非常高   | 非常高      |

> redis支持的类型比memcache多

##使用场景
* 缓存(********可以做mybatis的二级缓存)
* 计数器应用
* 网站需要统计注册用户数，网站总浏览次数
* 新浪微博转发数，点赞数
(能自增，自减)

* 实时防攻击系统：
* 防暴力破解：试密码

* 排行榜
* 设定有效期(自动过期)
* 自动去重应用(set数据结构)
* 队列
* 秒杀
* 消息订阅系统

## 结构

* redis-benchmark.exe ：性能测试工具
* redis-cli.exe ： 客户端
* redis-server.exe : 服务端
* redis-check-aof.exe : 检测持久化文件完好
* redis.windows.conf : 配置文件
